# Klasterisasi Provinsi di Indonesia Berdasarkan Angka Partisipasi Murni (APM) Pendidikan Menggunakan Algoritma K-Means

## Deskripsi Proyek

Aplikasi ini digunakan untuk melakukan klasterisasi provinsi di Indonesia berdasarkan Angka Partisipasi Murni (APM) pendidikan pada jenjang:

- SD/Sederajat
- SMP/Sederajat
- SMA/Sederajat

Metode yang digunakan adalah **K-Means Clustering** untuk mengelompokkan provinsi berdasarkan kemiripan tingkat partisipasi pendidikan.

Hasil klasterisasi dapat membantu dalam mengidentifikasi provinsi dengan tingkat partisipasi pendidikan tinggi, sedang, dan rendah sehingga dapat mendukung pengambilan keputusan dan penyusunan kebijakan pendidikan yang lebih tepat sasaran.
